## Cuentas para el elemento 0 y 9

### Elemento 0 (*)

```
011|Bienes y valores entregados
021|Derechos sobre instrumentos financieros
031|Otras cuentas de orden deudoras
041|Contrapartida de cuentas de orden deudoras
061|Bienes y valores recibidos
071|Compromisos sobre instrumentos financieros
081|Otras cuentas de orden acreedoras
091|Contrapartida de cuentas de orden acreedoras
```

* (*) Cuentas recomendadas en el PCGE 2019.
