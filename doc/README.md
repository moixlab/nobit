# Proyecto Nobit

## Origen del proyecto

1. Permitir tener un sistema de registro de información para la toma decisiones con datos relevantes.
2. Optimizar el consumo de recursos de computo y energéticos a lo mínimo posible que permita su transmisión a gran distancia.
3. Ser distribuido y portable, con resistencia a fenómenos previstos en los Andes.
4. Priorizar para la contabilidad del sector agrícola en Perú, con enfoque en la normativa para Micro y pequeñas empresas (MYPES).

## Características

- [ ] Guardado en archivos en una MicroSD
- [ ] Utilizar SQLite
- [ ] Utilizar KV
